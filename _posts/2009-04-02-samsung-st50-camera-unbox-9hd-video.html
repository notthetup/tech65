More gadgets coming in to Tech65! This time it's Samsung's latest consumer camera, the ST50. For a start, it's 12.2 megapixels, 3x optical zoom. It's not designed for the same audience as the WB500 we reviewed the last time, but it seems to be quite user friendly. Oh, and this is the first device we've played with at Tech65 that features Samsung's new unified user interface.
